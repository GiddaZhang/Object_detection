0.之前遇到的问题
  巨大框，直接框了一片
  小框，框住了人体的部分，一般就是头或者脚
  

1.目前尝试效果较好的方法
  切割一次，调用nms，再delete
  切割两次，把结果合并到一块，调用nms，再delete
  切割一次，对结果里面反常大框重新检测，得到的是原有+二次新检测出来的框放到一起，再delete，nms
